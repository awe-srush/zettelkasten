# Ops Meta-Metrics

In addition to tracking bugs, we should also track:

- Changes (who/what/when/type)
- Incidents (Type/Severity)
- Response to incidents (TTR/TTD)

How do you have confidence in infrastructure?

- Measure it
- graph it
- alert on it
